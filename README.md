# input-draw

`input-draw` is a custom HTML element that allows users to draw using their mouse or touch screen.

## Demo
- https://code4fukui.github.io/input-draw/

## Features
- Extends the HTMLElement to create a custom drawing element
- Supports mouse and touch input
- Encoded drawing data is available as a data protocol

## Usage
```html
<script type="module" src="https://code4fukui.github.io/input-draw/input-draw.js"></script>
<input-draw color=red pen-width=20 style="display:block;border:1px solid black;width:100%;height:80vh;" id=draw></input-draw>
```
